# OCT 助手

![OCT 图标](../assets/oct_icon.png)

**OpenClaw Terminal 配置和使用技能**

---

## 🎯 简介

这个技能提供 OCT (OpenClaw Terminal) 的配置模板和使用指南。

OCT 是一个桌面应用，让你通过自然对话控制电脑、处理任务、获取信息。

**核心理念**：一个会思考的终端窗口，你说它做。

---

## 📦 下载安装

### 方式 1：通过 ClawHub 安装

```bash
clawhub install oct-helper
```

### 方式 2：下载 OCT 桌面应用

**GitHub Release**: https://github.com/zl585451/openclaw-terminal/releases/tag/v0.1.2

| 平台 | 安装包 | 大小 |
|------|--------|------|
| Windows | OCT-Setup-v0.1.2.exe | 79.7 MB |
| Mac Intel | OCT-0.1.2-Mac-x64.dmg | 101 MB |
| Mac M1/M2 | OCT-0.1.2-Mac-arm64.dmg | 95.8 MB |
| Linux | OCT-0.1.2-Linux-x86_64.AppImage | 105 MB |

## 🆕 核心功能

| 功能 | 说明 |
|------|------|
| 💬 智能对话 | 和 AMY 聊天，表格优先，结构化展示 |
| 🧠 思维引导 | 弹出选择题帮你理清思路 |
| 🎮 系统控制 | 文件操作、应用管理、系统设置 |
| 🔧 开发助手 | 代码生成、Git 操作、终端命令 |
| 📊 信息获取 | 天气查询、新闻摘要、数据搜索 |

---

## 🚀 快速开始

1. 安装 OCT 桌面应用
2. 启动 OCT，开始对话
3. 输入 `/help` 查看可用命令
