## 🎯 OCT 是什么？

OCT 是 OpenClaw 的**桌面客户端**，让你用聊天的方式控制电脑、处理任务、获取信息。

**简单说**：一个会思考的终端窗口，你说它做。

---

## 💖 核心理念

### 📊 友好的对话式表格

**OCT vs 传统社交软件**

| 对比项 | 飞书/QQ/微信 | OCT |
|--------|-------------|-----|
| **消息展示** | 纯文本刷屏 | 结构化表格 |
| **信息密度** | 低（需要滚动） | 高（一目了然） |
| **AGENT 回复** | 长篇大论刷屏 | 表格 + 要点总结 |
| **交互方式** | 点击 + 输入 | 自然对话 |
| **学习成本** | 需要熟悉界面 | 会说话就会用 |

**表格一目了然，再也不用被 AGENT 刷屏！**

## 🚀 快速开始

### 1️⃣ 安装 OCT
下载 `OCT-Setup-v0.1.0.exe`，双击运行，按提示安装

### 2️⃣ 启动 OCT
双击桌面快捷方式或运行：
```bash
E:\windows-window\OpenClaw-Terminal\启动.bat
```

### 3️⃣ 连接 Gateway
确保 OpenClaw Gateway 已启动（自动或手动）

### 4️⃣ 开始对话
在输入框输入消息，按 Enter 发送

## 🎮 常用命令

| 命令 | 功能 |
|------|------|
| `/status` | 查看当前会话状态 |
| `/new` | 开启新会话 |
| `/restart` | 重启 Gateway |
| `/subagents` | 查看后台任务 |
| `/cot on/off` | 开启/关闭思维链 |
