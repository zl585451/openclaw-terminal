# OpenClaw Terminal (OCT)

**Windows 超级系统助手 · 你的 AI 桌面终端**

---

## 🎯 OCT 是什么？

OCT 是 OpenClaw 的**桌面客户端**，让你用聊天的方式控制电脑、处理任务、获取信息。

**简单说**：一个会思考的终端窗口，你说它做。


提醒：本地电脑安装有OPENCLAW  并且openclaw安装在windows环境下 如果是在wsl环境下AGENT要越系统操作 体验没有那么好   如果没有gateway 本软件就是个AI聊天机器人  不能执行任何系统任务

## 💖 核心理念

### 📊 友好的对话式表格

**OCT vs 传统社交软件**

| 对比项 | 飞书/QQ/微信 | OCT |
|--------|-------------|-----|
| **消息展示** | 纯文本刷屏 | 结构化表格 |
| **信息密度** | 低（需要滚动） | 高（一目了然） |
| **AGENT 回复** | 长篇大论刷屏 | 表格 + 要点总结 |
| **交互方式** | 点击 + 输入 | 自然对话 |
| **学习成本** | 需要熟悉界面 | 会说话就会用 |

## 🚀 快速开始

### 1. 启动 OCT
双击桌面快捷方式或运行：
```bash
E:\windows-window\OpenClaw-Terminal\启动.bat
```

### 2. 连接 Gateway
确保 OpenClaw Gateway 已启动（自动或手动）

### 3. 开始对话
在输入框输入消息，按 Enter 发送

## 🎮 常用命令

| 命令 | 功能 |
|------|------|
| `/status` | 查看当前会话状态 |
| `/new` | 开启新会话 |
| `/restart` | 重启 Gateway |
| `/subagents` | 查看后台任务 |
| `/cot on/off` | 开启/关闭思维链 |

## 💡 使用技巧

### ✅ 高效用法
- 用自然语言描述需求，不用记命令
- **表格优先**：能表格就不刷屏
- 复杂任务自动 spawn 子代理后台执行
- 多任务并行，不阻塞对话

### ⚡ 快捷键
| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Enter` | 发送消息 |
| `Ctrl+Shift+S` | 截图 |
| `↑/↓` | 输入历史 |
| `Ctrl+C` | 复制选中内容 |

### 🎯 最佳实践
- 日常对话：关闭 CoT，简洁回复
- 调试问题：开启 CoT，看思考过程
- 批量任务：用子代理，后台执行
