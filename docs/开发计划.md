## 🔌 集成方案

### 1. REAPER 控制集成

**方案 A: OSC 协议（推荐）**
- REAPER 原生支持 OSC 控制
- Node.js 发送 OSC 消息
- 低延迟，稳定

**方案 B: ReaScript**
- REAPER 脚本（EEL/Python/Lua）
- 功能强大，但需要脚本环境

**方案 C: UI 自动化**
- 模拟鼠标键盘操作
- 不推荐（不稳定）

**推荐**: 方案 A（OSC 协议）

---

### 2. AI.library 集成

**现有能力**:
- `audio_knowledge_search` - 搜索知识库
- `audio_knowledge_retrieve` - 检索具体内容

**集成方式**:
- OCT 调用 OpenClaw 工具
- AMY 自动判断何时查询知识库
- 结果直接显示在聊天中

---

### 3. OpenClaw 技能融合

**已有技能**:
- 飞书通道 ✅
- REAPER 技能（规划中）
- AI.library 工具 ✅

**融合方式**:
```
用户："帮我把这段音频降噪"
  ↓
AMY 理解意图
  ↓
调用 REAPER 技能（打开 REAPER → 创建音轨 → 加载降噪插件）
  ↓
调用 AI.library（查询降噪参数建议）
  ↓
返回结果到 OCT 聊天
```
