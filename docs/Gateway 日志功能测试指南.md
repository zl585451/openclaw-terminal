
### 9.5 Gateway 启动
**测试方法**:
1. 确保 Gateway 未运行
2. 点击 `▶ 启动` 按钮
3. 观察日志区

**预期结果**:
- ✅ Gateway 进程启动
- ✅ 按钮变为 `■ 停止`
- ✅ 日志区开始显示启动日志
- ✅ 连接状态变为 `CONNECTED`

**测试步骤**:
```
1. 关闭 Gateway（或重启 OCT）
2. 点击 "▶ 启动" 按钮
3. 等待 2-3 秒
4. 观察日志区是否有启动日志
5. 确认顶部状态显示 CONNECTED
```

## 🐛 常见问题排查

### 问题 1: 日志区一直显示"等待 Gateway 日志..."
**可能原因**:
- Gateway 未启动
- 日志监控未正确初始化

**解决方法**:
```
1. 点击 "▶ 启动" 按钮
2. 如果已经是外部 Gateway，显示"● 已连接"
3. 重启 OCT
```

### 问题 2: 导出按钮无响应
**可能原因**:
- 日志区为空
- 浏览器阻止下载

**解决方法**:
```
1. 确保日志区有内容
2. 检查浏览器下载权限
3. 查看浏览器下载拦截提示
```

### 问题 3: 日志颜色不显示
**可能原因**:
- 日志格式不标准
- CSS 样式未加载

**解决方法**:
```
1. 检查日志格式是否为 [标签] 内容
2. 刷新 OCT 窗口
3. 查看开发者工具 Console 是否有错误
```
