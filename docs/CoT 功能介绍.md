# OpenClaw CoT · 思维链功能

**让 AI 思考过程可视化，更透明、更可控**

---

## 🎯 什么是 CoT？

**CoT = Chain of Thought（思维链）**

简单说，就是让 AI 把思考过程展示给你看，而不是只给一个最终答案。

---

## ✨ 核心特性

### 📖 思考过程可见
```
用户：这个代码为什么报错？

AI 思考中...
1. 先检查语法错误 → 无
2. 检查变量定义 → 发现未声明
3. 检查依赖导入 → 缺失模块
4. 给出解决方案 → 添加 import 语句

✅ 问题已解决！
```

### 🎮 一键开关
- **开启 CoT**: 查看 AI 思考过程
- **关闭 CoT**: 只看最终结果

### ⚡ 零学习成本
不需要学习任何命令，界面操作，点一下就行。

---

## 🚀 使用场景

| 场景 | 效果 |
|------|------|
| **调试代码** | 看 AI 如何分析问题 |

## 🎮 如何开启/关闭

### 方法 1：快捷按钮
点击界面右上角的 **💭** 按钮

### 方法 2：命令
```
开启：/cot on
关闭：/cot off
查看：/cot status
```

## 💡 小贴士

- ✅ **建议开启**: 学习、调试、复杂问题
- ⚡ **建议关闭**: 日常聊天、简单问答
